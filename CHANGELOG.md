# Lịch sử cập nhật HP Cons PDF

## v1.0.9
- Thanh tab: mở nhiều file không còn mất file đầu — có nút "Danh sách" liệt kê tất cả file đang mở để nhảy nhanh.
- Kéo tab qua lại giữa các cửa sổ (tách ra hoặc gộp về cửa sổ khác).
- Copy trang giữa các cửa sổ (clipboard trang dùng chung).
- Chuyển tab bằng Ctrl+Tab / Ctrl+Shift+Tab (đã bỏ cuộn chuột đổi tab để tránh nhầm).

## v1.0.8
- Sửa hiện tượng đứng máy khi chuyển trang trên bản vẽ nặng.
- Thêm chữ: chọn phông chữ, cỡ (chiều cao) chữ, màu chữ và màu nền.
- Hộp thoại in ghi nhớ thiết lập lần trước (số bản, màu, khổ giấy, cách in mặt, hướng, tỉ lệ).
- Che trắng theo màu nền — khớp nền file scan.
- Kéo/sửa ghi chú (chữ/hình mình thêm) ngay bằng công cụ Xem, không cần chuyển sang Chọn.

## v1.0.7
- Xóa vùng và di chuyển được cả trên file mà máy đọc sai tọa độ (bản vẽ CAD xuất lỗi).
- Công cụ Chọn: kéo để khoanh chọn cả vùng, bấm để chọn một đối tượng.
